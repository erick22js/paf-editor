Animation

  • A => Select all objects
  • V => Enter in drag mode
  • T => Enter in translate mode
  • S => Enter in scale mode
  • K => Enter in skew mode
  • R => Enter in rotate mode
  • B => Enter in selection mode
  • X => Lock into X-Axis
  • Y => Lock into Y-Axis
  • F9 => Play current animation
  • Ctrl+C => Copy selected objects actions
  • Ctrl+V => Paste selected objects actions
  • Arrow Left => Select previous frame
  • Arrow Right => Select next frame
  • Shift+Arrow Left => Select first frame
  • Shift+Arrow Right => Select last frame
  • Alt+Arrow Left => Retroceds frame in scroll
  • Alt+Arrow Right => Advance frame in scroll
  • Ctrl+Arrow Left => Retroceds 10 frames
  • Ctrl+Arrow Right => Advances 10 frames
